# Free PC Games Hub

A static website demo for a free PC games library with a landing page, game list, dynamic detail page, and simple admin draft panel.

## Files

- `index.html` — landing page
- `games.html` — game list loaded from `games.json`
- `game-page.html` — game details page with URL query parameter support
- `admin.html` — demo admin panel saving game draft data in localStorage
- `style.css` — shared site styles
- `script.js` — dynamic game list rendering
- `games.json` — game metadata

## How to use

1. Open `index.html` in a browser.
2. Visit `games.html` to see available game cards.
3. Click a game to open its detail page.
4. Use `admin.html` to save a draft locally.

## Notes

- The download links and images are placeholders.
- Replace `https://your-shortlink.com/...` with real links for production.
- For live updates, connect the site to a backend or CMS.
